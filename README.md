Ren'Py Narrative Systems Sample

This sample demonstrates a small interactive "Moment" similar to the structure described in the job listing.

Features:
- branching dialogue
- conditional responses based on emotional state
- memory-driven narrative changes
- relationship state variables
- debug overlay for testing routes

Key systems:
tone
trust
mem_pos
mem_pain

These variables influence dialogue and available choices.

Debugging:
Press F2 to open the debug overlay and modify states.
